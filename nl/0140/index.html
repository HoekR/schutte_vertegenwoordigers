Falckenburch, Robbrecht

Nr. 140 1632-? agent agent Q. Luik Pag. 186
Functie
agent
Categorie
Luik
Periode
1632–?

140. Robbrecht Falckenburch ontving 20 dec. 1632 zijn commissie als agent te Luik. Een naamgenoot van hem wordt sedert 1605 als agent der Staten Generaal vermeld; deze zal begraven zijn te 's-Gravenhage(Gr.k.)9jan. 1632.
